# Statistical Modeling Portfolio  
Predicting Breast Cancer Diagnosis   

## Overview  
This repository contains my STA 631 Statistical Modeling Portfolio, developed using the Breast Cancer Wisconsin Diagnostic Dataset. The project demonstrates applied statistical modeling, model comparison, and workflow development using R and the tidymodels framework.

## View Portfolio  
https://nancy-odhiambo.github.io/Statistical-Modeling-Potfolio/

## Methods Implemented  
- Logistic Regression (full, reduced, polynomial, interaction)  
- Lasso and Ridge Regression  
- Linear Discriminant Analysis (LDA)  
- Multinomial Logistic Regression  
- Cross‑validation, ROC analysis, and performance metrics  

## Tools  
- R, tidyverse, tidymodels  
- glmnet, MASS, nnet  
- GGally, corrplot, kableExtra  

## Purpose  
This portfolio demonstrates reproducible modeling workflows, model selection, and interpretation of classification models in a health‑data context.
